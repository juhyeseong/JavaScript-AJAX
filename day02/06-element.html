함수 매개변수 반환형 설명
getElementById 문자열 형태의 id 단일 요소 id를 지정하여 문서 요소를 불러온다
querySelector 문자열 형태의 css 선택자 단일 요소 css 선택자를 이용하여 첫번째 요소를 불러온다
querySelectorAll 문자열 형태의 css 선택자 여러 요소(NodeList) css 선택자를 이용하여 조건에 맞는 모든 요소를 불러온다
함수 매개변수 반환형 설명
createElement 문자열 형태의 태그 이름 HTMLelement 지정한 태그 요소를 생성한다. 문서에 반영되지 않는다
A.appendChild(B) HTMLelement HTMLelement (B) A요소의 하위 B요소를 마지막에 추가한다
A.removeChild(B) HTMLelement HTMLelement (B) A요소의 하위에서 B요소를 찾아서 제거한다
함수 매개변수 반환형 설명
A.getAttribute 문자열 형태의 속성 이름 문자열 형태의 속성 값 A 태그요소에서 B이름의 속성을 찾아서 그 값을 반환한다
A.setAttribute 순서대로 속성이름, 속성값 A 태그요소에 name으로 value를 추가한다
A.removeAttribute(name) 문자열 형태의 속성이름 A태그요소에서 name과 일치하는 속성을 제거한다
